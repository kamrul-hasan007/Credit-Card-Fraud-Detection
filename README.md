# Credit-Card-Fraud-Detection

📊 Credit Card Data Preprocessing Project

This project focuses on data preprocessing using a credit card transaction dataset. The main objective is to clean and prepare raw data by handling common issues such as missing values, duplicate records, invalid entries, and outliers.

Various preprocessing techniques were applied, including data cleaning, transformation, and normalization to improve data quality. The final dataset is well-structured and ready for further analysis or machine learning tasks.

🔧 Key Steps:

Introduced real-world data issues (missing values, duplicates, invalid data)
Handled missing values using mean imputation
Removed duplicate records
Fixed invalid values
Detected and removed outliers (IQR method)
Converted data types and normalized features

📌 This project demonstrates a complete data preprocessing workflow in R.
